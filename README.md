# QCK-SSG
release 0.1
# Features
To start the app: 

```npm install```

To create .html files.
## One

```node . -i index.txt``` (index.txt as an example you can use another file name )

## Many

```node . -i /src/pages``` (/src/page is an example you can use another path to a folder )


# Version

```node . -- version (-v) ```

# Help

```node . -- help (-h)```
